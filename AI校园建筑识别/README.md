# AI校园建筑识别系统

## 项目简介

AI校园建筑识别系统是一个基于深度学习的实时物体识别应用，能够通过摄像头实时识别校园内的建筑和垃圾，并提供垃圾分类建议。

## 核心功能

- 实时摄像头画面捕获与处理
- 校园建筑和垃圾的实时识别
- 垃圾分类建议生成
- 友好的用户界面

## 技术栈

- **前端**：HTML5 + JavaScript + Bootstrap
- **后端**：Python 3.9+ + Flask
- **AI框架**：PyTorch + Ultralytics YOLOv8
- **图像处理**：OpenCV

## 环境搭建

1. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

2. **准备数据集**
   - 将校园建筑照片和垃圾图像放入 `data/images/` 目录
   - 使用 LabelImg 工具对图像进行标注，生成 YOLO 格式的标注文件
   - 将标注文件放入 `data/labels/` 目录

3. **下载预训练模型**
   - 下载 YOLOv8n 预训练模型并放入 `models/` 目录

## 模型训练

1. **配置数据集**
   - 修改 `data/config.yaml` 文件，确保路径和类别设置正确

2. **运行训练脚本**
   ```bash
   python train.py
   ```

3. **训练完成后**
   - 最佳模型将保存到 `models/best.pt`

## 运行应用

1. **启动 Flask 应用**
   ```bash
   cd app
   python app.py
   ```

2. **访问应用**
   - 打开浏览器，访问 `http://localhost:5000`
   - 点击"开始识别"按钮，系统将开始实时识别

## 使用说明

- **开始识别**：点击"开始识别"按钮，系统将启动摄像头并开始实时识别
- **停止识别**：点击"停止识别"按钮，系统将停止识别
- **切换摄像头**：点击"切换摄像头"按钮，系统将切换前后摄像头（开发中）
- **查看结果**：识别结果将显示在"识别结果"区域，包括物体类别、置信度和垃圾分类建议

## 技术难点及解决思路

1. **校园建筑识别**
   - 问题：现有模型对特定校园建筑识别准确率低
   - 解决思路：收集更多校园建筑样本，进行数据增强，微调预训练模型

2. **实时性能**
   - 问题：移动设备上实时识别性能受限
   - 解决思路：使用轻量级模型（YOLOv8n），模型量化，硬件加速

3. **环境光照变化**
   - 问题：不同光照条件下识别准确率波动
   - 解决思路：数据增强（模拟不同光照），图像预处理（自适应亮度调整），鲁棒性训练

## 后续优化方向

- 模型持续迭代优化
- 增加更多校园建筑和垃圾类型
- 支持离线识别
- 集成到校园APP中

## 注意事项

- 首次运行时，系统会自动下载 YOLOv8 预训练模型
- 训练模型需要一定的计算资源，建议在 GPU 环境下运行
- 实时识别可能会消耗较多的设备资源，建议在性能较好的设备上运行
